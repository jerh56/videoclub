Enunciado del problema

Se desea hacer una aplicación para obtener información de los juegos y películas de un videoclub.

El videoclub maneja solamente dos tipos de productos: películas y juegos.

Para todos los productos se tiene la siguiente información: nombre del producto, 
la cantidad a pagar por rentar ese producto, cantidad de días que se presta el producto 
cuando se renta y si en ese momento el producto está o no rentado.

Para cada una de las películas se tiene el género (acción, fantasía, drama, aventura, etc.) y
 el año de producción. Para los juegos se cuenta con el estilo (acción, deportes, aventuras, etc.) 
 y la plataforma (Xbox, Playstation, Wii) en la que se puede ejecutar. La aplicación debe mostrar la
 información de las películas y juegos, así como establecer la renta o devolución de uno de los productos.
